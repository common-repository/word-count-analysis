A simple but useful plugin that gives the word count of your articles.

== Description ==

A plugin that shows how many words the content in your articles or pages consists of and the number of unique words (stopwords).

It makes a simple but functional analysis of your articles. It calculates words, such as the number of words and characters in your sentences. Calculates the longest and shortest word count for each post and gives an average value.

You can also see the articles of your authors in a table.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `word-count-analysis` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You should perform the first analysis by clicking Word C&A from the left menu.
4. In the form that appears, you must select the language of your web page.
5. Click the Calculate button and wait for it to calculate. This period may be shorter or longer depending on your total writing.
6. When the calculation is finished, the page will be refreshed and you can see the analysis tables.
7. You can do this again by going to the Recalculate page.


== Frequently Asked Questions ==

= Does the plugin count words correctly? =
If there are not many different special characters in your text, it counts 95% correctly. If you think it is wrong, you can contact us.

= Does it work other than the specified languages? =

It will only do word count, but it may not work correctly.

= Whats StopWords? =

You can learn more from this source. [StopWords Wiki](https://en.wikipedia.org/wiki/Stop_word)
